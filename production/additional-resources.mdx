The guides in this documentation cover the patterns and decisions specific to the Bulletproof React architecture. The resources below go further — they address the underlying principles that make those patterns work. Whether you are deepening your React knowledge, filling gaps in your JavaScript foundations, or learning how to write cleaner code, these links are worth your time.

<Note>
Focus on understanding principles rather than memorizing tool-specific APIs. The ecosystem changes quickly, but the ideas behind component composition, state management, and clean code transfer across frameworks and libraries.
</Note>

## React resources

These resources cover React from official documentation through community-developed philosophy and patterns.

<CardGroup cols={2}>
  <Card title="Official documentation" icon="book-open" href="https://react.dev/">
    The authoritative source for React concepts, APIs, and hooks. The new docs site includes interactive examples and updated guidance for modern React patterns.
  </Card>
  <Card title="Tao of React" icon="compass" href="https://alexkondov.com/tao-of-react/">
    A concise guide to the principles and mental models behind well-structured React applications, covering component design, state, and project organization.
  </Card>
  <Card title="React Handbook" icon="book" href="https://reacthandbook.dev/">
    A practical reference covering common React patterns, hooks usage, and real-world application structure with clear code examples.
  </Card>
  <Card title="React Philosophies" icon="lightbulb" href="https://github.com/mithi/react-philosophies">
    A curated list of ideas and observations about writing good React code, synthesizing community wisdom on when and how to apply React features.
  </Card>
  <Card title="React Patterns" icon="layout" href="https://reactpatterns.com/">
    A collection of common React component patterns with concise examples to help you recognize and apply well-established solutions to recurring problems.
  </Card>
  <Card title="React TypeScript Cheatsheet" icon="code" href="https://react-typescript-cheatsheet.netlify.app/">
    A community-maintained reference for using TypeScript with React, covering prop types, hooks, context, generics, and common gotchas.
  </Card>
</CardGroup>

## JavaScript resources

A solid JavaScript foundation makes React work easier and more intuitive. These resources address the language itself.

<CardGroup cols={2}>
  <Card title="You Don't Know JS" icon="book-marked" href="https://github.com/getify/You-Dont-Know-JS">
    A free book series that explores JavaScript's core mechanisms in depth, including scope, closures, `this`, prototypes, and async behavior.
  </Card>
  <Card title="JavaScript.info" icon="info" href="https://javascript.info/">
    A comprehensive, well-structured tutorial covering modern JavaScript from the basics through advanced topics, with interactive exercises throughout.
  </Card>
  <Card title="33 concepts every JavaScript developer should know" icon="list-ordered" href="https://github.com/leonardomso/33-js-concepts#8-iife-modules-and-namespaces">
    A structured learning guide mapping 33 fundamental JavaScript concepts to curated articles, videos, and books for each topic.
  </Card>
  <Card title="JavaScript to know for React" icon="react" href="https://kentcdodds.com/blog/javascript-to-know-for-react">
    Kent C. Dodds' focused breakdown of the specific JavaScript features — destructuring, spread, template literals, modules, and more — you need to be effective with React.
  </Card>
</CardGroup>
